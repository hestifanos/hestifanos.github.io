Maze Creater and Solver

About Maze Solver Application

Welcome to the Maze Solver Application! This application allows you to generate and solve mazes using various algorithms such as Depth-First Search (DFS) and A* (A Star) search.

Features:

  1. Maze Generation: You can generate mazes of custom sizes using the Randomized Prim's algorithm. Simply input the width and height of the maze, and click the "Generate Maze" button.
  2. Maze Solving: Once you have generated a maze, you can solve it using either Depth-First Search (DFS) or A* (A Star) search algorithms. The application will visually guide you through the solution process.
  3. Interactive Interface: You can interact with the maze canvas to set the start and end points for solving the maze. Click on any cell to set the start or end point.

How to Use:

  1. Generate Maze: Input the desired width and height of the maze, then click the "Generate Maze" button.
  2. Solve Maze: After generating a maze, choose either "Solve Maze with DFS" or "Solve Maze with A Star" to solve the maze using the selected algorithm.
  3. Interactive Solving: While the maze is being solved, you can step through the solution process using the "Previous Step" and "Next Step" buttons to observe each step of the algorithm.
  4. Adjust Settings: You can adjust settings such as animation speed and algorithm choice in the future to customize your solving experience.

Technologies Used:

  • Flask: Python web framework used for backend development.
  • HTML/CSS/JavaScript: Frontend technologies for building the user interface and interactivity.
  • jQuery: JavaScript library used for DOM manipulation and AJAX requests.
  • Canvas API: HTML5 canvas element used for drawing the maze and solution path.
